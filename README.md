# Teste Prático - Iniflex

Projeto desenvolvido em Java com base nos requisitos propostos no teste técnico.

## Funcionalidades
- Cadastro de funcionários
- Remoção de funcionário
- Aumento salarial
- Agrupamento por função
- Ordenação e filtros
- Cálculos com BigDecimal

## Tecnologias
- Java
- Programação Orientada a Objetos
- Streams API
